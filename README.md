# Career Survival Kit AI

Career Survival Kit AI is an AI-powered job application assistant designed to help students and early-career professionals maximize their chances of securing interviews.

By combining resume analysis, ATS optimization, interview preparation, recruiter outreach generation, and AI-powered career coaching into a single workflow, the platform provides a complete job-search support system rather than a standalone resume checker.

## Features

### ATS Compatibility Analysis

* Resume vs Job Description matching
* ATS compatibility scoring
* Strength and weakness identification
* Visual match indicators

### Keyword Gap Analysis

* Matched skills detection
* Missing keyword identification
* Priority skill recommendations
* Requirement coverage insights

### Resume Optimization

* Resume bullet enhancement
* STAR methodology improvements
* Action verb recommendations
* ATS-friendly content suggestions
* Export-ready optimized content

### Interview Preparation Toolkit

* Resume-based interview questions
* Job description-specific technical questions
* Behavioral interview preparation
* Revision topic recommendations
* Interactive interview simulation with feedback

### Recruiter Outreach Generator

* Personalized cold emails
* LinkedIn connection messages
* Referral request templates
* Informational networking outreach drafts

### AI Career Coach

* Context-aware Gemini-powered assistant
* Resume-aware responses
* Job description-aware recommendations
* Career guidance and interview support

## How It Works

1. Upload Resume (PDF/DOCX)
2. Paste Target Job Description
3. Generate Career Survival Kit
4. Review ATS Analysis
5. Identify Missing Skills
6. Optimize Resume Content
7. Practice Interview Questions
8. Generate Recruiter Outreach Messages
9. Interact with the AI Career Coach

## Why This Project

As a student actively applying for internships and technical roles, I repeatedly faced challenges in tailoring resumes, identifying missing skills, preparing for interviews, and creating professional outreach messages.

Career Survival Kit AI was built to consolidate these fragmented tasks into a single AI-powered workflow that helps candidates move from application to interview readiness faster.
